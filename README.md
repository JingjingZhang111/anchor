Anchor
===
It is a Novel Alignment-Free Method Based on the anchor, The program is written in python.

Usage
----
Usage
----
python anchor.py --seqs example.fasta --k 4 --epsilon 13 --savename results.meg

Input Parameter
---
--k the kmer length

--epsilon the space of kmer

--seqs  the seqs file (all sequence in one fasta file)

Output
---
--Distance matrix in meg format
